# 99 Nights in the Forest — Project Docs

This folder contains Markdown files you can drop into your project repo.

## Files
- **GDD-99-Nights.md** — Full Game Design Document tailored for Hytopia.
- **Implementation-Roadmap.md** — SDK-oriented build plan with file tree, schemas, and TypeScript skeletons.
- **Scaffold-Prompt.md** — One-shot prompt to generate the entire project scaffold in Cursor/Claude Code.
- **READ-FIRST.md** — Quick start and how to use each file.

## Quick Start
1. Open **Scaffold-Prompt.md**, copy everything inside the code block, and paste into Cursor or Claude Code.
2. Apply the generated files. Commit to your repo.
3. Use **Implementation-Roadmap.md** to wire systems and extend content.
4. Keep **GDD-99-Nights.md** as the north star for design decisions.
